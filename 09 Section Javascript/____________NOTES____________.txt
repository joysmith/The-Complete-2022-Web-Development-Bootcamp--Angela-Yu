 	var name = "Angela";			// store Angela in name box
	var name = name.toUpperCase();		// perform some functionality on name then, update the name variable 
	
	-> We can perform functionality on any piece of data but we have to update it in js, if dont do it the data will be lost 
	   so first line is to assign and then the second line is doing two things
	   a.  name.toUpperCase();  			perform functionality, note: this piece of data will be lost if its not saved somewhere
	   b.  var name = name.toUpperCase();		update the piece of info in name var with the new piece of info
